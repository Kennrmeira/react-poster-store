# 🎬 RockPosters - Loja de Pôsteres de Filmes e Séries

Projeto desenvolvido com **React + Vite** que simula uma pequena loja de pôsteres decorativos de filmes e séries.

A aplicação possui uma navegação simples entre páginas utilizando **React Router** e exibe cards de produtos contendo imagem, título, descrição e botão de compra.

Este projeto foi criado com objetivo de **praticar conceitos de desenvolvimento front-end**, estruturação de componentes e organização de aplicações React.

---

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript (ES6+)
- React Router
- HTML5
- CSS3 (inline styles)

## 🖥️ Funcionalidades

- Página inicial com opções de navegação
- Página de pôsteres de **Filmes**
- Página de pôsteres de **Séries**
- Exibição de produtos em formato de **card**
- Estrutura de componentes reutilizáveis
- Navegação entre páginas utilizando **React Router**
